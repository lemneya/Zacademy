# WhatsApp Registration Flow — Zidni Academy

## Overview

This is the registration system for the 7-day free AI challenge and the 28-day AI Skills Certificate.

**Channels:**
- WhatsApp (primary) — for registration, reminders, and communication
- Google Form (backup) — for structured data collection
- Facebook Messenger (optional) — for Facebook-native users

---

## Registration Options

### Option 1: WhatsApp Direct (Primary)

**How it works:**
1. User sends "AI" or "أريد التسجيل" to WhatsApp number
2. Auto-reply sends welcome message + registration form link
3. User fills Google Form
4. Confirmation message sent via WhatsApp
5. User added to WhatsApp broadcast list for daily challenge tasks

**WhatsApp number:** [TO BE SET — add actual number when available]

### Option 2: Google Form Direct

**Form fields:**
- Full name (الاسم الكامل)
- Phone number (رقم الهاتف)
- Neighborhood in Nouadhibou (الحي في نواذيبو)
- Occupation (المهنة)
- Why do you want to learn AI? (لماذا تريد تعلم الذكاء الاصطناعي?)
- Have you used ChatGPT before? (هل استخدمت ChatGPT من قبل?)
- Preferred language for lessons (اللغة المفضلة للدروس)
  - Arabic (العربية)
  - French (الفرنسية)
  - Hassaniya (الحسانية)
- Consent to receive WhatsApp messages (الموافقة على استلام رسائل واتساب)

### Option 3: Facebook Lead Form

For Facebook ad campaigns. Same fields as Google Form, integrated with Facebook Lead Ads.

## Daily Challenge Messages (7 Days)

### Day 1 Message
```
يوم 1 — تحدي زدني أكاديمي 🚀

مهمة اليوم: افتح ChatGPT واسأله 5 أسئلة عن نواذيبو أو عملك.

الخطوات:
1️⃣ ادخل على chatgpt.com
2️⃣ أنشئ حساباً مجانياً
3️⃣ اسأل 5 أسئلة
4️⃣ احفظ لقطة الشاشة لأفضل إجابة

عند الانتهاء، أرسل "تم" مع لقطة الشاشة.

سؤال للتفكير: ما هي أفضل إجابة حصلت عليها؟
```

### Day 2 Message
```
يوم 2 — تحدي زدني أكاديمي 🚀

مهمة اليوم: جرب صيغة السؤال الذهبية.

صيغة السؤال القوي = 3 أجزاء:
1️⃣ السياق: من أنت؟
2️⃣ المهمة: ماذا تريد؟
3️⃣ الصيغة: كيف تريد الإجابة؟

جرب هذا:
"أنا [عملك] من [حيك] في نواذيبو. اكتب لي منشوراً على فيسبوك لـ [منتجك] بسعر [السعر]. اكتب بالعربية، محلي وودي."

أرسل "تم" مع المنشور الذي كتبه الذكاء الاصطناعي.
```

### Day 3 Message
```
يوم 3 — تحدي زدني أكاديمي 🚀

مهمة اليوم: استخدم الذكاء الاصطناعي لعملك.

اختر واحداً:
أ) اكتب رسالة واتساب لزبون
ب) اكتب وصف منتج
ج) اكتب رداً على شكوى

استخدم ChatGPT. أرسل "تم" مع ما كتبته.
```

### Day 4 Message
```
يوم 4 — تحدي زدني أكاديمي 🚀

مهمة اليوم: انشر منشوراً على فيسبوك.

استخدم الذكاء الاصطناعي لكتابة منشور. انشره على صفحتك.

أرسل "تم" مع لقطة الشاشة للمنشور + عدد الإعجابات.
```

### Day 5 Message
```
يوم 5 — تحدي زدني أكاديمي 🚀

مهمة اليوم: اكتب رسالة واتساب احترافية.

اختر موقفاً:
- سؤال عن السعر
- تأخير في التوصيل
- شكوى من زبون

استخدم الذكاء الاصطناعي لكتابة رد قوي.

أرسل "تم" مع الرسالة.
```

### Day 6 Message
```
يوم 6 — تحدي زدني أكاديمي 🚀

مهمة اليوم: جرب أداة جديدة.

اختر واحدة:
- Copilot (copilot.microsoft.com)
- Gemini (gemini.google.com)
- Perplexity (perplexity.ai)

جربها لمهمة واحدة. قارنها مع ChatGPT.

أرسل "تم" مع: أي أداة جربتها، وهل كانت أفضل؟
```

### Day 7 Message
```
يوم 7 — تحدي زدني أكاديمي 🚀

المهمة الأخيرة: ابنِ مساعدك الشخصي.

اكتب نموذج مساعد شخصي مخصص لك (انظر اليوم 7 في الدورة).

اختبره بـ 3 أسئلة.

أرسل "تم" مع مساعدك الشخصي + أفضل إجابة.

---

🎉 مبروك! أكملت تحدي 7 أيام!

ستصلك شهادة إتمام خلال 48 ساعة.

هل تريد الاستمرار في الدورة الكاملة (28 يوماً)؟
أرسل "دورة" للتفاصيل.
```

## Technical Setup

### WhatsApp Business API (Recommended for scale)
- Use WhatsApp Business API via Meta Business Partners
- Or use WhatsApp Business App for small scale
- Set up automated welcome messages
- Use broadcast lists for daily challenge messages

### Google Form Setup
- Create form at forms.google.com
- Link to Google Sheet for responses
- Set up notifications for new responses
- Export responses daily

### Automation (Optional)
- Zapier: Google Form response → WhatsApp message
- Make.com: Alternative to Zapier
- Google Apps Script: Free automation within Google ecosystem

- Always ask for consent before adding to broadcast lists
- Respect WhatsApp's anti-spam policies
- Keep messages short and actionable
