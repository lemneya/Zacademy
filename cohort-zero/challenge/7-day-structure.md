### Day 1 — What is AI?

**Lesson Objective:** Understand what AI is and isn't, create a ChatGPT account, and run first prompts.

**Morning Message (8:00 AM):**

```
📅 يوم 1 — تحدي زدني أكاديمي 🚀

الموضوع: ما هو الذكاء الاصطناعي؟

ببساطة: الذكاء الاصطناعي آلة تتعلم من الأمثلة،
مثلما يتعلم صياد في نواذيبو تمييز السمك الطازج من غير الطازج بعد سنوات من الخبرة.

❌ ليس سحراً
❌ ليس خطيراً
✅ أداة عملية مثل الكمبيوتر والإنترنت
```

**Activity:**
```
المهمة:
1️⃣ افتح chatgpt.com على هاتفك أو كمبيوترك
2️⃣ أنشئ حساباً مجاناً (يطلب إيميل فقط)
3️⃣ اسأل ChatGPT 5 أسئلة عن نواذيبو أو عملك

أمثلة:
- "اكتب لي 5 حقائق عن مدينة نواذيبو"
- "اشرح لي كيف أبدأ مشروع صغير في نواذيبو"
- "ترجم هذي الجملة للإنجليزية"

عند الانتهاء: أرسل "تم" + أفضل إجابة حصلت عليها
```

**Homework:**
```
الواجب:
جرّب سؤالاً واحداً عن مجالك:
- إذا كنت تاجر: "كيف أحسن خدمة الزبائن؟"
- إذا كنت طالب: "شرح مبسط لمصطلح علمي"
- إذا كنت صاحب محل: "أفكار لزيادة المبيعات"

خذ لقطة شاشة لأفضل إجابة وأرسلها في المجموعة.
```

**Engagement Question (Evening 7:00 PM):**
```
سؤال المساء 🤔

ما ��و الشيء الذي تمنيت أن الذكاء الاصطناعي يعمله لك؟
اكتب إجابتك في المجموعة 👇
```

### Day 2 — Prompting

**Lesson Objective:** Write effective prompts using the 3-part formula.

**Morning Message (8:00 AM):**

```
📅 يوم 2 — تحدي زدني أكاديمي 🚀

الموضوع: كيف تكتب سؤالاً قوياً؟

معظم الناس يسألون أسئلة ضعيفة ويحصلون على إجابات ضعيفة.
السر في 3 أجزاء:

1️⃣ السياق — من أنت؟ (أنا تاجر في سوق السمك)
2️⃣ المهمة — ماذا تريد؟ (اكتب لي منشور فيسبوك)
3️⃣ الصيغة — كيف تريد الإجابة؟ (بالعربية، ودية، 100 كلمة)
```

**Activity:**
```
المهمة:
استخدم الصيغة الذهبية لكتابة 3 أوامر قوية.

جرب هذا القالب:
"أنا [دورك/عملك] من [حيك] في نواذيبو.
[المهمة التي تريدها].
[كيف تريد الإجابة]."

مثال:
"أنا صاحب محل مواد غذائية في حي العيشيش.
اكتب لي منشور فيسبوك عن تخفيضات نهاية الأسبوع.
بالعربية، ودية، 80 كلمة، مع دعوة لزيارة المحل."

أرسل "تم" مع أفضل أمر كتبته + النتيجة
```

**Homework:**
```
الواجب:
اكتب أمراً قوياً لموقف حقيقي في حياتك.
طبقه على ChatGPT.
صحح الأمر إذا كانت النتيجة سيئة.
أرسل النسخة النهائية.
```

**Engagement Question (Evening 7:00 PM):**
```
سؤال المساء 🤔

هل لاحظت الفرق بين سؤال عادي وسؤال قوي؟
ما هو السؤال الذي أعجبك أكثر اليوم؟
```

### Day 3 — AI for Business

**Lesson Objective:** Apply AI to real business communication — WhatsApp messages and product descriptions.

**Morning Message (8:00 AM):**

```
📅 يوم 3 — تحدي زدني أكاديمي 🚀

الموضوع: الذكاء الاصطناعي لأعمالك

أتذكر أول مرة أرسلت رسالة لزبون وما عرفت كيف تصيغها؟
اليوم، الذكاء الاصطناعي يكتبها لك في ثوانٍ.

3 تطبيقات مباشرة:
✅ رسائل واتساب احترافية للزبائن
✅ وصف منتجات مقنع
✅ ردود على استفسارات وشكاوى
```

**Activity:**
```
المهمة:
اختر واحداً من التالي:

أ) اكتب رسالة واتساب لزبون تسأله إذا كان راضياً عن المنتج
ب) اكتب وصف منتج لشيء تبيعه (حتى لو كانت فكرة فقط)
ج) اكتب رداً على شكوى من زبون غير راضٍ

استخدم الصيغة من أمس:
"أنا [دوري]. سياقي هو [الموقف]. اكتب [الرسالة]."

أرسل "تم" مع الرسالة التي كتبها الذكاء الاصطناعي
```

**Homework:**
```
الواجب:
جرّب موقفاً حقيقياً:
افتح واتساب، شوف آخر محادثة مع زبون أو مورد.
استخدم ChatGPT لكتابة رد أفضل.
أرسل لقطة شاشة للمقارنة: ردك القديم vs رد ChatGPT.
```

**Engagement Question (Evening 7:00 PM):**
```
سؤال المساء 🤔

أي نوع من الرسائل التجارية تكتبها أكثر؟
- رسائل للزبائن
- تواصل مع الموردين
- إعلانات ومنشورات
- شكاوى واستفسارات

شاركنا تجربتك 👇
```

### Day 4 — Social Media

**Lesson Objective:** Use AI to create Facebook content that engages local audiences.

**Morning Message (8:00 AM):**

```
📅 يوم 4 — تحدي زدني أكاديمي 🚀

الموضوع: منشورات فيسبوك بالذكاء الاصطناعي

فيسبوك هو السوق الرقمي لنواذيبو.
منشور جيد = زبائن جدد.

مكونات المنشور الناجح:
1️⃣ عنوان يلفت الانتباه
2️⃣ فقرة قصيرة مفيدة
3️⃣ دعوة للتفاعل (سؤال، تعليق، مشاركة)
4️⃣ صورة أو فيديو (إذا أمكن)
```

**Activity:**
```
المهمة:
استخدم ChatGPT لكتابة منشور فيسبوك.

اكتب أمراً مثل:
"أنا صاحب [نشاطك التجاري] في نواذيبو.
اكتب لي منشوراً فيسبوكياً عن [منتجك/خدمتك].
بالعربية، محلي، ودود. في آخره سؤال للتفاعل. 80 كلمة."

انشر المنشور على صفحتك أو بروفايلك.
أرسل "تم" مع لقطة شاشة للمنشور.
```

**Homework:**
```
الواجب:
انشر المنشور فعلياً (حتى لو صفحتك الشخصية).
أرسل رابط المنشور أو لقطة شاشة.
لاحظ كم إعجاب وتعليق حصلت عليه خلال 24 ساعة.
```

**Engagement Question (Evening 7:00 PM):**
```
سؤال المساء 🤔

شاركنا رابط المنشور الذي نشرته اليوم.
كم إعجاب حصل عليه؟
هل كتبت مثل هذا المنشور من قبل بدون الذكاء الاصطناعي؟
```

### Day 5 — Customer Service

**Lesson Objective:** Handle difficult customer situations professionally using AI.

**Morning Message (8:00 AM):**

```
📅 يوم 5 — تحدي زدني أكاديمي 🚀

الموضوع: التعامل مع الزبائن الصعبين

أصعب شيء في التجارة: الرد على زبون غاضب أو غير راضٍ.

الذكاء الاصطناعي يساعدك:
✅ يهدئ الزبون بلغة مهذبة
✅ يقدم حلولاً عملية
✅ يحافظ على علاقتك معه

الأمر السحري:
"أنا [دوري]. زبون غير راضٍ بسبب [السبب]. اكتب رداً مهذباً يعتذر ويقدم حلاً."
```

**Activity:**
```
المهمة:
اختر موقفاً:

أ) زبون اشتكى من تأخير في التوصيل
ب) زبون غير راضٍ عن جودة المنتج
ج) زبون يطلب استرداد المال

استخدم ChatGPT لكتابة رد محترف.
أرسل "تم" مع الرد الذي كتبه ChatGPT
```

**Homework:**
```
الواجب:
خذ موقفاً حقيقياً من تجربتك (أو من تجربة شخص تعرفه).
اكتب أمراً لـ ChatGPT بناءً على الموقف الحقيقي.
أرسل النتيجة + هل كانت مفيدة؟
```

**Engagement Question (Evening 7:00 PM):**
```
سؤال المساء 🤔

هل سبق أن خسرت زبوناً بسبب رد سيء؟
كيف كان يمكن للذكاء الاصطناعي أن يساعدك؟
شاركنا قصتك 👇
```

### Day 6 — Tool Exploration

**Lesson Objective:** Try an AI tool beyond ChatGPT to understand the ecosystem.

**Morning Message (8:00 AM):**

```
📅 يوم 6 — تحدي زدني أكاديمي 🚀

الموضوع: أدوات ذكاء اصطناعي أخرى

ChatGPT ليس الوحيد. هناك أدوات أخرى مجانية وكلها مفيدة:

🧠 Copilot (copilot.microsoft.com)
   — من Microsoft، مدمج مع بحث Bing، مجاني

🧠 Gemini (gemini.google.com)
   — من Google، يقدر يقرأ الصور، مجاني

🧠 Perplexity (perplexity.ai)
   — متخصص في البحث، يذكر المصادر، مجاني

كل أداة لها قوتها. اليوم تجرب أداة جديدة.
```

**Activity:**
```
المهمة:
اختر أداة واحدة من الثلاثة:
1️⃣ Copilot
2️⃣ Gemini
3️⃣ Perplexity

جربها لنفس المهمة التي جربتها مع ChatGPT يوم 1.
قارن النتائج.

أرسل "تم" مع:
- أي أداة جربتها
- هل كانت أفضل أم أسوأ من ChatGPT؟
- لماذا؟
```

**Homework:**
```
الواجب:
جرّب أداة مختلفة عن التي اخترتها في النشاط.
أي أداة ستفيدك أكثر في مجالك؟
```

**Engagement Question (Evening 7:00 PM):**
```
سؤال المساء 🤔

بعد تجربة 4 أدوات (ChatGPT، Copilot، Gemini، Perplexity):
أي واحدة ستحتفظ بها في هاتفك؟
ولماذا؟
```

### Day 7 — Personal Toolkit

**Lesson Objective:** Build a reusable AI assistant customised to the participant's own life and work.

**Morning Message (8:00 AM):**

```
📅 يوم 7 — آخر يوم في التحدي 🚀

الموضوع: بناء مساعدك الشخصي بالذكاء الاصطناعي

اليوم تبني شيئاً يبقى معك للأبد:
مساعد شخصي مخصص لك (Custom GPT / Instructions).

الفكرة:
تكتب تعليمات دائمة لـ ChatGPT عن نفسك وعملك.
بعدها، كل مرة تسأله، يجاوب بناءً على هويتك.

مثال:
"أنت مساعدي الشخصي. أنا تاجر سمك في سوق نواذيبو.
ساعدني في: كتابة رسائل للزبائن، حساب الأسعار، متابعة الديون.
استخدم العربية. كن عملياً ومختصراً."
```

**Activity:**
```
المهمة:
ابنِ مساعدك الشخصي.

1️⃣ اكتب فقرة تصف فيها:
   - من أنت
   - ماذا تفعل
   - كيف تريد المساعدة
   - اللغة والطريقة

2️⃣ أرسلها لـ ChatGPT كـ "تعليمات دائمة"
   (في ChatGPT: انقر على اسمك > Custom Instructions)

3️⃣ اختبر مساعدك بـ 3 أسئلة

أرسل "تم" مع المساعد الذي بنيته + أفضل إجابة
```

**Homework:**
```
الواجب النهائي (اختياري لكن قوي):
شارك مساعدك الشخصي مع شخص آخر في المجموعة.
جرب مساعده واسأله سؤالاً.
سترى كيف تختلف المساعدات حسب الشخص.
```

**Engagement Question (Evening 7:00 PM):**

```
🎉🎉🎉 مبروك!

أكملت تحدي 7 أيام.

أنت الآن:
✅ تفهم ما هو الذكاء الاصطناعي
✅ تكتب أوامر قوية
✅ تستخدم ChatGPT و Copilot و Gemini و Perplexity
✅ تكتب منشورات ورسائل احترافية
✅ تتعامل مع الزبائن باحترافية
✅ عندك مساعد شخصي خاص بك

هذا فقط البداية.

الدورة الكاملة (28 يوماً) تشمل:
📚 28 درساً عملياً
🎓 شهادة مهارات الذكاء الاصطناعي
👥 مجتمع خريجي زدني أكاديمي
💼 مشروع نهائي يضاف لسيرتك الذاتية

💰 السعر: 500 أوقية فقط (مرة واحدة)
📲 أرسل "دورة" للتسجيل الآن

الموارد الطبيعية قد تنتهي، لكن المعرفة لا تنتهي.
```
